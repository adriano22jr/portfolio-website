# 🌐 Personal Portfolio Website

This is the repository for my **personal portfolio website**!  
It showcases my projects, technical skills, and experiences, and serves as my online presence for professional connections.

## 🚀 Features
- **Responsive design** with Bootstrap components.  
- **Homepage** with personal introduction.  
- **Projects section** with descriptions and GitHub links.  
- **Skills & Experience section** to highlight my technical background.  
- **Contact section** with links to email and social profiles.  

## 🛠️ Built With
- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)  
- [Bootstrap](https://getbootstrap.com/)  
- [GitHub Pages](https://pages.github.com/) for hosting
